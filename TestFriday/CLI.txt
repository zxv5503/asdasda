Emp클래스를 생성하여 변수를 선언하였습니다.
ArrayList를 이용해 Emp클래스의 생성자에 사용자가 입력한 값들을 저장할 수 있도록 설계하였습니다.

case 1 :
emp에서가져온 정보들을 list에 담아서 향상된for문을 이용해 데이터정보를 출력합니다.

cse 2 :
스캐너를 통해 사용자가 값을 직접 입력할 수 있게 하였습니다.

case3 :
회원이름을 입력하여 데이터를 삭제할 수 있도록 하였습니다.
데이터를 정수로 선언하여 값이 바뀔경우에 성공, 바뀌지않을 경우에 실패 문구가 뜨도록 설계하였습니다.

case4:
회원이름을 입력하여 데이터값을 검색할 수 있도록 하였습니다.
do-while문을 이용하여 존재하지 않는 데이터가 입력되더라도 경고문을 출력할 수 있게 하였습니다.

case5:
사원번호를 입력하여 데이터값을 수정할 수 있도록 하였습니다.
if문을 이용하여 성공,실패,검색 실패와 같은 안내문을 출력하도록 하였습니다.

case 6:
시스템을 종료시킵니다.

dafault를 통해 case 선택상황에서 1~6번을 제외한 값을 입력시 올바른 값을 입력하라는 문구가 뜨도록 하였습니다.
